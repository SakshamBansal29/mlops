Login to docker nd docker hub
Build docker image: docker build -t docker-demo
Run the container: docker run -p 5000:5000 docker-demo
Tag the image: docker tag docker-demo sakshambansal29/docker-demo:latest
Push image to docker hub: docker push sakshambansal29/docker-demo:latest
Pull image from dcoker docker pull sakshambansal29/docker-demo:latest 
Run the pulled docker image: docker run -p 5000:5000 sakshambansal29/docker-demo:latest
